# gitHub 프로젝트와 이슈

1. 프로젝트 생성
2. GitKraken에 연동
3. Issues : 업무
   1. Label : 업무 종류
      1. enhance : 개발
      2. document : 기획, 보고서
   2. milestone : 일정표
4. pull request : 합치기
5. Actions : 빌드 배포 자동화
6. Projects : 애자일 소프트웨어 개발법
7. settings : 자동화, 설정, 인증
8. workflows : 자동화
9. discussion : 토의하는 곳, setting에서 추가해야 됨
10. sprint : 주 단위로 업무 구분하기, 새로 추가해야 됨
